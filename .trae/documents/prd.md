# 综合业务分析系统 — 前端产品需求文档（PRD）

## 1. 产品概述

云梦县综合业务分析系统前端，面向政府领导及各部门工作人员，以大屏驾驶舱为首页入口，提供视频监控、AI行为分析、预警处置、城管协同、应急指挥、门牌管理、一标三实核采、网格化管理、报表统计等功能的可视化操作界面。本阶段仅开发前端，所有数据使用模拟数据，不涉及后端。

**目标用户**：县级领导、城管执法部门、公安派出所、社区网格员、各乡镇基层工作人员

## 3. 核心功能模块

### 3.1 综合驾驶舱（首页大屏）

**页面定位**：系统首页，面向县级领导展示全县治理核心指标

| 模块名称 | 功能描述 |
|----------|----------|
| 设备概况 | 全县摄像头总数、在线数、离线数、在线率（环形图） |
| 实时事件 | 滚动展示最新预警事件列表（事件类型、时间、地点、级别） |
| 预警趋势 | 近7天/30天预警数量折线图 |
| 处置率统计 | 各部门预警处置率柱状图 |
| AI分析概况 | 当前分析任务数、今日分析事件数、各类型占比饼图 |
| 城管案件概况 | 今日案件数、待处置数、已结案数 |

| 网格概况 | 网格总数、在岗网格员数、今日事件上报数 |

| 快捷入口 | 各功能模块快捷跳转按钮 |

## 4. 核心流程

### 4.1 预警处置流程

用户进入预警中心 → 查看实时预警列表 → 点击预警查看详情 → 派发至处置人 → 处置人反馈处置结果 → 核查确认 → 闭环归档

```mermaid
flowchart TD
    "AI分析触发事件" --> "事件分流"
    "事件分流" --> "城管事件"
    "事件分流" --> "其他预警事件"
    "城管事件" --> "城管协同模块"
    "其他预警事件" --> "预警中心"
    "预警中心" --> "接收预警"
    "接收预警" --> "派发工单"
    "派发工单" --> "处置反馈"
    "处置反馈" --> "核查闭环"
    "城管协同模块" --> "案件派遣"
    "案件派遣" --> "处置核查"
    "处置核查" --> "结案归档"
    "核查闭环" --> "报表统计"
    "结案归档" --> "报表统计"
```

### 4.2 一标三实核采流程

创建核采任务 → 分配网格员 → 移动端采集 → 数据校验 → 比对核验 → 入库关联 → 常态化更新

```mermaid
flowchart TD
    "创建核采任务" --> "分配至网格员"
    "分配至网格员" --> "移动端信息采集"
    "移动端信息采集" --> "数据质量校验"
    "数据质量校验" --> "与公安库比对"
    "与公安库比对" --> "差异核查"
    "差异核查" --> "数据入库关联"
    "数据入库关联" --> "常态化动态更新"
```

## 5. 用户界面设计

### 5.1 设计风格

- **主色调**：深蓝色系（#0A1628 背景 + #1B3A5C 卡片 + #00D4FF 科技蓝强调色），营造政务大屏科技感
- **辅助色**：绿色（正常/在线）、橙色（预警/待处理）、红色（告警/紧急）、紫色（数据/分析）
- **字体**：标题使用 DIN Alternate / Orbitron（数字指标），正文使用 PingFang SC / Microsoft YaHei
- **布局**：左侧导航栏 + 顶部面包屑 + 主内容区，大屏为全屏沉浸式布局
- **图标**：线性图标风格，统一使用 Lucide Icons
- **动效**：数字滚动动画、图表渐入、卡片悬浮微动、预警闪烁

### 5.2 页面设计概览

| 页面名称 | 模块名称 | UI要素 |
|----------|----------|--------|
| 综合驾驶舱 | 设备概况 | 环形进度图 + 数字滚动动画，深色卡片 |
| 综合驾驶舱 | 实时事件 | 滚动列表，预警级别色标，自动滚动 |
| 综合驾驶舱 | 预警趋势 | 折线图，渐变填充区域 |
| 综合驾驶舱 | GIS地图 | 深色底图，设备/事件标注点聚合 |
| 视频资源管理 | 设备列表 | 表格 + 筛选栏，状态标签彩色 |
| 视频资源管理 | 实时预览 | 宫格布局，视频占位图，PTZ控制面板 |
| 行为分析 | 实时分析监控 | 视频画面 + AI标注框叠加效果 |
| 预警中心 | 预警列表 | 表格 + 级别筛选标签，闪烁高亮新预警 |
| 城管业务协同 | 案件详情 | 左侧信息卡 + 右侧流程时间线 |
| 应急指挥调度 | 资源一张图 | 全屏GIS地图 + 浮动资源面板 |
| 门牌管理 | 标准地址库 | 树形层级导航 + 地址表格 |
| 一标三实核采 | 数据关联融合 | 关系图谱可视化（力导向图） |
| 网格化管理 | 网格划分 | GIS地图 + 网格多边形叠加 |
| 报表中心 | 自定义报表 | 拖拽维度选择器 + 实时图表更新 |

### 5.3 响应式设计

- 桌面优先（1920×1080 为基准分辨率）
- 大屏页面支持 3840×2160 4K输出
- 最小支持 1366×768 笔记本屏幕
- 侧边栏可折叠适配窄屏

## 6. 模拟数据说明

所有数据均为前端模拟，不依赖后端API：

| 数据类别 | 模拟内容 |
|----------|----------|
| 设备数据 | 200+ 条摄像头记录（含位置、状态、类型） |
| 预警事件 | 50+ 条预警记录（各类型、各级别） |
| 城管案件 | 30+ 条案件记录（各状态） |
| 标准地址 | 100+ 条地址记录（七段码） |
| 一标三实 | 人口50条、房屋30条、单位20条 |
| 网格数据 | 15个网格、20名网格员 |
| 统计数据 | 各类图表所需的聚合数据 |
